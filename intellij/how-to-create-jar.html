How to Create a Jar File in IntelliJ IDEA

Once your program works and you are ready to create a Jar file:

Click File, Project Structure, then on the left side click Artifacts.

Click the green Plus symbol, then select JAR, then From modules with dependencies...

Beside Main Class, click the button with the 3 dots.

Click the Project tab, then expand your project folder, then src, then your package, then select your program and click OK.

Beside Directory for META-INF/MANIFEST.MF:, add a folder to the path for your manifest file. For example:

     change: C:\Users\2459037845\Documents\Java\Workspace\HelloWorld\src

     to: C:\Users\2459037845\Documents\Java\Workspace\HelloWorld\src\HelloWorldMF

Click OK.

In the top field beside Name:, change the name of the artifact, if desired. This will be the name you select when you want to build the .jar file.

In the middle pane, just above the META_INF folder, you will see the filename of your jar. If you wish to change it, right click it and select  Rename. Change the name and then press ENTER.

Click  OK.

Now, back on the main screen of IntelliJ:

Click  Build, then  Build Artifacts, select your artifact name, then click  Build.

Once it is finished, your Jar file will be in the  out/artifacts/[artifact name]/ folder.

Right click your Jar file and select  Run to make sure it works.
